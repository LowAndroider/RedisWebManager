# 接口
## 通用返回格式
```json
{
  "code": 200,
  "msg": "请求成功",
 
  "data": ""
}
```
### 登录
#### url： /redisLogin
#### 请求方式：GET
#### 参数
| 参数| 说明| 必填 | 类型| 示例|
|---|----|---|---|---|
|host| ip或者网址 | 是 | String | 127.0.0.1|
|port| 端口号| 否 | int| 6379(默认值)|
|auth| 密码| 否 | String|123456|
#### 响应数据
```json
{
  "code": 200,
  "msg": "请求成功"
}
```
